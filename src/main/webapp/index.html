Currency correlations on Forex

What is currency correlation?

Correlation coefficient is the measure of relationship between assets. If two currencies are correlated, they will move in the same direction in Forex (their prices tend to change similarly).

What does this graph represent?

This is Minimum Spanning Tree showing correlations between selected currencies, for a given time span. It displays clusters of highly connected assets.
